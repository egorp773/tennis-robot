# Обучение YOLO на теннисные мячи

## Общий план

```
Фотографируешь → Загружаешь в Roboflow → Размечаешь →
Экспортируешь → Запускаешь train.py → Загружаешь модель на Pi
```

---

## Шаг 1 — Что и как фотографировать

### Сколько фото
Минимум **300**, лучше **500–800**. Больше = точнее.

### Обязательные сцены

| Сцена | Зачем | Примерно |
|-------|-------|----------|
| 1 мяч, 0.5–1м от камеры | Крупный план | 50 фото |
| 1 мяч, 2–4м от камеры | Средний план | 80 фото |
| 1 мяч, 5–10м от камеры | Мелкий, как в реальности | 80 фото |
| 3–5 мячей в кадре | Несколько целей | 60 фото |
| 10+ мячей разбросаны | Много целей | 40 фото |
| Мяч у белой линии | Сложный контраст | 40 фото |
| Мяч в тени | Другое освещение | 30 фото |
| Мяч частично закрыт ракеткой/ногой | Окклюзия | 20 фото |
| **Пустой корт без мячей** | Учит не ошибаться | 80 фото |

### Углы съёмки
- **Главное** — снимай **с высоты камеры робота** (~30–50 см от земли, угол ~30° вниз).
  Это самый важный угол — именно так камера будет видеть мячи.
- Дополнительно: с высоты ~1м, сверху ~90°

### Освещение
- Яркое солнце (резкие тени)
- Облачно (мягкий свет)
- Вечер/утро (низкое солнце)
- Если есть — искусственный свет в зале

### Положения мяча
- На корте в разных зонах (у сетки, у задней линии, в центре)
- На линии разметки
- Вблизи стойки сетки
- Несколько мячей кучкой

### Что НЕ нужно
- Фото в помещении (если корт уличный)
- Размытые фото
- Фото где мяч вообще не виден

## Шаг 2 — Roboflow: загрузка и разметка

### Создание проекта
1. Зайди на **roboflow.com** → Sign Up (бесплатно)
2. Create Project → выбери **Object Detection**
3. Название: `tennis-ball`, annotation group: `ball`

### Загрузка фото
1. Upload → перетащи все фото
2. Roboflow сам разобьёт на train/val/test (80/15/5)

### Разметка (самое долгое — ~2–4 часа)
1. Открой каждое фото
2. Нарисуй **прямоугольник** вокруг каждого мяча
3. Класс: `tennis_ball`
4. Фото без мячей — просто нажми "Save" без разметки (это и есть hard negatives)

**Советы по разметке:**
- Рамка должна плотно обтягивать мяч
- Не включай тень в рамку
- Частично видимый мяч — всё равно размечай

### Аугментации (Roboflow сделает сам)
После разметки → Generate → включи:
- Flip Horizontal ✓
- Rotation: ±15° ✓
- Brightness: -25% to +25% ✓
- Blur: до 1px ✓
- Noise ✓

Это увеличит датасет в 3–5 раз автоматически.

### Экспорт
1. Export Dataset → Format: **YOLOv8**
2. Выбери **"download zip to computer"**
3. Распакуй в папку `training/dataset/`

Структура должна быть:
```
training/dataset/
  data.yaml
  train/
    images/   ← фото
    labels/   ← txt разметка
  valid/
    images/
    labels/
  test/
    images/
    labels/
```

## Шаг 3 — Обучение

### Требования
```bash
pip install ultralytics
```

GPU (видеокарта) сильно ускоряет обучение. Если нет GPU — используй Google Colab (см. ниже).

### Запуск
```bash
cd C:\robot\tennis\training
python train.py
```

Обучение займёт:
- С GPU (RTX 3060+): ~20–40 минут
- Без GPU (CPU): 3–8 часов
- Google Colab (бесплатный GPU): ~30–60 минут

### Google Colab (если нет GPU)
1. Открой `train_colab.ipynb` в Jupyter или загрузи на colab.research.google.com
2. Runtime → Change runtime type → **GPU**
3. Загрузи `dataset.zip` когда скрипт попросит
4. Скачай `best.pt` по окончании

---

## Шаг 4 — Загрузка модели на Pi

```bash
cd C:\robot\tennis
python upload_model.py
```

Скрипт сам скопирует `best.pt` на Pi и обновит `main.py` чтобы использовать новую модель.
