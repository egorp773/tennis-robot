# Compiled Wiki

This folder stores cleaned, stable project knowledge.

Use it for:

- Final architecture summaries.
- Validated test conclusions.
- Accepted calibration procedures.
- Hardware setup guides.
- Reliable troubleshooting guides.

Rules:

- Keep entries concise and factual.
- Link back to raw material when useful.
- Update `MEMORY.md`, `DECISIONS.md`, or `docs/ai_skills/` when compiled knowledge changes how Codex should work.

Compiled material is intended for AI and humans to read before making project decisions.
