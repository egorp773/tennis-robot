# Raw Wiki

This folder stores raw project material.

Use it for:

- Chat exports.
- Test dumps.
- Uncleaned notes.
- Ideas.
- Sensor logs.
- Screenshots descriptions.
- Temporary investigation notes.

Rules:

- Do not aggressively rewrite raw material.
- Prefer appending new raw notes over editing history.
- If a raw note becomes stable knowledge, summarize it into `docs/wiki/compiled/` or `docs/ai_skills/`.

Raw material is not the final source of truth.
