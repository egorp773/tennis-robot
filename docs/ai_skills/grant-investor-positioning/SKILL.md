# Grant Investor Positioning

## Positioning

Present this as a hardware-integrated prototype moving toward safe autonomous tennis-ball collection.

Emphasize:

- Working mobile app control.
- ESP32 robot controller.
- Raspberry Pi vision pipeline.
- UWB localization architecture.
- Safety-first roadmap.
- Clear future path to more anchors and trained model.

## Avoid Overclaiming

Do not claim:

- Production readiness.
- Full-court autonomous reliability.
- Ambiguity-free two-anchor UWB.
- Proven real-court YOLO accuracy before tests.

## Honest Readiness

Current state:

- Manual control: usable prototype.
- Autonomous mode: requires UWB tracker and safety improvements.
- Demo readiness: possible only with constrained demo mode.
- Product readiness: not yet.

## Strong Demo Story

The best grant/investor demo is:

- App connects.
- Working area and anchors are shown.
- Robot is calibrated.
- One zone is selected.
- Robot moves safely to that zone.
- Camera detects a ball.
- Robot stops or approaches slowly.
- Safety and telemetry are visible.

This demonstrates integration without pretending full product autonomy is finished.
